# AI-Powered Phishing Detection Platform

## Overview
A machine learning-based web application that detects phishing websites using a Random Forest classifier.

## Features
- Phishing website detection
- Flask web interface
- Confidence score prediction
- Interactive cybersecurity dashboard

## Tech Stack
- Python
- Flask
- Scikit-Learn
- Pandas
- HTML/CSS

## Model Performance
Accuracy: 96.7%

## How to Run

pip install -r requirements.txt
python app.py
